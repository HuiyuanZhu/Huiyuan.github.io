基于Node.js 的AngularJS 开发环境搭建

发表于 2018-12-31 | 更新于 2019-01-05 | 分类于 前端开发 ， Node.js ， AngularJS
  1. 下载安装Node.js，输入node -v查看是否安装成功。因为新版的Node.js已经集成了npm，所以无需另外下载。npm -v查看npm是否安装成功。

  2. 安装Angular CLI并创建新项目，该链接为官方网站，可以在该网站上查看相关命令行。
    常用命令：

    • 安装Angular CLI：npm install -g @angular/cli，安装完成后可用ng version或者ng v查看版本号。
    • 创建新项目：ng new 项目名称。
    • 编译构建项目，保存在“project”路径下，缺省则在“dist/”中：ng build [project]
    • 创建components、routes、services和pipes：ng generate。
    • 本地测试项目，默认路径为http://localhost:4200/ ：ng serve
    • 单元测试：ng test
    • End-to-end测试：ng e2e

    angular-cli vs @angular/cli:

    • @angular/cli是新版本，它的命名和Angular 2的模块命名一致，比如：@angular/common和@angular/router。
    • @angular/cli 需要Node 6.9.0或者以上版本，但是angular-cli 在Node 4或者以上版本即可，所以@angular/cli 在Node 4没法工作。
  3. 运行ng serve之后在浏览器打开http://localhost:4200/ 能出现以下网站即说明项目创建成功。
