Markdown简介

Markdown 是一种轻量级标记语言（英语：Lightweight Markup Language，简称LML），轻量级标记语言是一类用简单句法描述简单格式的文本语言。Markdown 容易书写，使得作者能更加专注于文章内容而非文本格式。
