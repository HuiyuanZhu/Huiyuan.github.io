Github - gist

发表于 2019-01-04 | 更新于 2019-01-05 | 分类于 Git ， Github

对于gist比较感兴趣的几个功能点：

  1. gist 支持markdown但是文件名后缀要为md。
  2. 可以对已经发布了的gist之前的版本利用Revisions按钮进行追踪，不用担心修改之后找不到之前的版本了。
  3. 可以利用JavaScript 将gist嵌入网页中。在WordPress中可以利用插件或者插入短代码。
  4. 利用Google Analytics 来监测访问量。
  5. 利用Gisto可在桌面管理并且筛选gist。
